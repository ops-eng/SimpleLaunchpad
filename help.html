Getting Started

  1. Open Preferences from the menu bar icon menu.
  2. Add apps in Manage Apps.
  3. Set your activation shortcut in Shortcut.

Common Tasks

  • Show or hide launcher
  • Reorder apps in the list
  • Delete selected apps
  • Adjust icon size, spacing, and columns

Troubleshooting Hotkey

Changes to shortcut take effect immediately.

  1. Check System Settings > Privacy & Security > Accessibility.
  2. Ensure SimpleLaunchpad is listed and enabled. You may need to add it and enable it.
  3. The hotkey might be in use by another application.
  4. Try a very unique shortcut combination for testing.
